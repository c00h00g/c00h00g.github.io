一些工作体会

发表于 2018-02-10   |   分类于 杂谈

工作总结

  1. 经常对自己提问。能问出好的问题才能说明对这个问题有自己深入对思考了
  2. 临时方案与理想方案。做事情的时候，不要只想到临时方案临时解决下问题，要想想理想方案是什么，否则，会导致系统越来越难维护
  3. 看到问题的本质。看到问题的时候，要分析下是不是一类问题，而不是单独case by case
  4. 问题分解。学会将问题分解，把复杂问题转化成小问题来做。
  5. 前瞻性。要想到后面要做什么，甚至可以从公司的角度去思考，应该做那些布局，三年后应该做成什么样子，倒逼自己进步。
