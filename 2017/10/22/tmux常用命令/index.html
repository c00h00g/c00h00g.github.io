tmux常用命令

发表于 2017-10-22   |   分类于 程序技巧

基本概念

  1. session
  2. window
  3. panel

主要命令

session

  1. 新建一个tmux session 

    直接在终端输入tmux
或者tmux new -s session_name

  2. 列出所有session

    tmux ls

  1. 加载一个session

    tmux a -t session_id

  2. 脱离当前session

    prefix + d

  3. 删除session

    tmux kill-session -t session_id

  4. 重命名session

    prefix + $
window

  1. 新建窗口

    prefix + c

  2. 重命名窗口

    prefix + ，

  3. 切换到具体某个窗口

    prefix + window_id

  4. 关闭当前窗口

    prefix + &
panel

  1. 上下分屏

    prefix + %

  2. 左右分屏

    prefix + ”

  3. 取消分屏

    prefix + x

  4. 如何滚屏

    prefix + pageUp/PageDown(mac上对应fn+上下键)

注意

  1. prefix指的是操作的快捷键ctrl + b
  2. 当然这些都是用到的一些最基础的操作，也可以通过修改.tmux.conf来自己配制
